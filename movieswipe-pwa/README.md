# MovieSwipe PWA 🎬

Progressive Web Application для выбора фильмов через интуитивный свайп-интерфейс и подбор по настроению.

## ✨ Основные возможности

- **Свайп-интерфейс** - быстрый выбор фильмов жестами (свайп вправо = добавить в избранное)
- **Подбор по настроению** - выбор из 12 настроений для персонализированных рекомендаций
- **Полный каталог** - поиск и фильтрация по жанрам, рейтингу, длительности
- **Избранное** - сохранение любимых фильмов с сортировкой
- **Детали фильмов** - полная информация о каждом фильме
- **Offline-first** - работает без интернета после первой загрузки
- **PWA** - устанавливается как нативное приложение

## 🚀 Технологии

- **React 18** + TypeScript
- **Zustand** - управление состоянием
- **Dexie.js** - IndexedDB для хранения данных
- **Workbox** - Service Worker и кэширование
- **React Router** - навигация
- **React Swipeable** - свайп-жесты
- **Vite** - сборка и разработка

### Генерация базы фильмов

Приложение поддерживает два способа загрузки фильмов:

#### 1. Предустановленные файлы по жанрам (рекомендуется)

Генерируйте JSON файлы с фильмами из TMDB API:

```bash
# Только базовый файл (100 фильмов, ~30 сек)
TMDB_API_KEY=your_key npm run generate:base

# Все жанры (8000+ фильмов, ~40-50 мин)
TMDB_API_KEY=your_key npm run generate:all

# Конкретный жанр
TMDB_API_KEY=your_key node scripts/generate-genre-files.js --genre=comedy
```

Подробнее: [QUICK_START.md](./QUICK_START.md)

#### 2. Динамическая загрузка через TMDB API

Пользователи могут добавить свой API ключ в настройках и загружать фильмы динамически.

Подробнее: [TMDB_GUIDE.md](./TMDB_GUIDE.md)

## 🌐 Интеграция с TMDB API

Приложение поддерживает загрузку полной базы фильмов из The Movie Database (TMDB).

### Получение API ключа

1. Зарегистрируйтесь на [themoviedb.org](https://www.themoviedb.org/signup)
2. Перейдите в [настройки API](https://www.themoviedb.org/settings/api)
3. Запросите API ключ (выберите "Developer" при регистрации)
4. Скопируйте "API Key (v3 auth)"

### Использование

1. Откройте **Настройки** в приложении
2. Включите чекбокс **IMDB / TMDB**
3. Вставьте ваш API ключ в поле ввода
4. Нажмите **Сохранить**
5. Нажмите **Обновить каталог**

Приложение загрузит до 1000 популярных фильмов с:
- Русскими названиями и описаниями
- Постерами и фоновыми изображениями
- Жанрами и рейтингами
- Автоматическим маппингом жанров на настроения

**Детальная информация** (режиссеры, актеры, точная длительность) загружается автоматически при открытии страницы фильма и кэшируется для offline доступа.

### Особенности загрузки

- **Прогресс-бар** показывает процесс загрузки
- **Постеры загружаются по требованию** - не включены в начальную загрузку
- **Данные сохраняются в IndexedDB** - работают offline после загрузки
- **Поиск работает по всей базе** - включая названия, актеров, режиссеров
- **Фильтрация по жанрам и настроениям** - автоматический маппинг из TMDB

### Ограничения TMDB API

- Бесплатный план: 40 запросов в 10 секунд
- Приложение использует **~50 запросов** для начальной загрузки 1000 фильмов
- Детали фильмов (режиссеры, актеры) загружаются по требованию при открытии (1 запрос на фильм)
- Загрузка 1000 фильмов занимает ~15-20 секунд
- Детали кэшируются в IndexedDB и не требуют повторной загрузки

## 🎬 Интеграция с ПоискКино API

Приложение поддерживает инкрементальную загрузку из базы **1,085,549 фильмов** через ПоискКино API (агрегатор Кинопоиск + TMDb + IMDb).

### Получение API ключа

1. Перейдите на [poiskkino.dev](https://poiskkino.dev/)
2. Нажмите "Получить API"
3. Следуйте инструкциям в Telegram боте @poiskkinodev_bot
4. Скопируйте ваш API ключ

### Использование скрипта синхронизации

```bash
# Первый запуск (загрузит до 50,000 фильмов)
node scripts/poiskkino-sync.cjs --api-key YOUR_KEY

# Ограничить количество запросов
node scripts/poiskkino-sync.cjs --api-key YOUR_KEY --max-requests 10

# Проверить статус
node scripts/poiskkino-sync.cjs --status

# Сбросить прогресс
node scripts/poiskkino-sync.cjs --reset
```

### Особенности ПоискКино

- ✅ **Русский язык** - названия, описания, актеры на русском
- ✅ **Умные фильтры** - рейтинг > 6.0, популярные, с постерами
- ✅ **Инкрементальная загрузка** - можно остановить и продолжить
- ✅ **Автоматический лимит** - отслеживает 200 запросов/день
- ✅ **Временная стратегия** - начинает с последних 5 лет, расширяется назад
- ✅ **Прогресс** - сохраняется в `.sync-progress.json`

### Лимиты ПоискКино API

- Бесплатный план: **200 запросов в сутки**
- Фильмов за запрос: до 250
- Максимум в день: ~50,000 фильмов
- Сброс лимита: каждый день в 00:00 UTC

### Автоматизация (cron)

```bash
# Добавить в crontab для ежедневной синхронизации
0 3 * * * cd /path/to/project/scripts && node poiskkino-sync.cjs --api-key YOUR_KEY >> sync.log 2>&1
```
