# Быстрая синхронизация с несколькими ключами

## Использование

### Вариант 1: Ключи через запятую
```bash
node poiskkino-sync-multi.cjs KEY1,KEY2,KEY3
```

### Вариант 2: Ключи через пробел
```bash
node poiskkino-sync-multi.cjs KEY1 KEY2 KEY3
```

## Примеры

### С двумя ключами
```bash
cd movieswipe-pwa/scripts
node poiskkino-sync-multi.cjs ABC123,XYZ789
```

### С тремя ключами
```bash
node poiskkino-sync-multi.cjs ABC123 XYZ789 DEF456
```

## Что происходит

Скрипт автоматически:
1. ✅ Запускает синхронизацию с первым ключом
2. ✅ Когда лимит исчерпан (200 запросов) - переключается на второй
3. ✅ Продолжает до тех пор пока не закончатся ключи или фильмы
4. ✅ Показывает итоговую статистику

## Преимущества

- 🚀 **Один запуск** - не нужно вручную переключать ключи
- ⚡ **Автоматически** - скрипт сам определяет когда переключиться
- 📊 **Статистика** - показывает сколько загружено каждым ключом
- 💾 **Безопасно** - прогресс сохраняется после каждого ключа
- ⏸️ **Можно прервать** - Ctrl+C в любой момент, прогресс сохранен

## Сколько фильмов загрузится?

С рейтингом > 5.0:
- **1 ключ:** ~80,000 фильмов
- **2 ключа:** ~160,000 фильмов
- **3 ключа:** ~240,000 фильмов
- **4 ключа:** ~320,000 фильмов

## Если прервать

Просто запусти снова с теми же или другими ключами:
```bash
# Продолжит с того же места
node poiskkino-sync-multi.cjs KEY1,KEY2
```

## Добавить еще ключей

Если ключи закончились, но синхронизация не завершена:
```bash
# Добавь новые ключи к существующим
node poiskkino-sync-multi.cjs KEY1,KEY2,KEY3,KEY4
```

Скрипт автоматически пропустит исчерпанные ключи и продолжит с новых!
