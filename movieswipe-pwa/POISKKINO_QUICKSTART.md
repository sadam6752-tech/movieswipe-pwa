# ПоискКино - Быстрый старт

Загрузите реальную базу из 1M+ фильмов на русском языке за 10 дней!

## Шаг 1: Получите API ключ

1. Откройте https://poiskkino.dev/
2. Нажмите "Получить API"
3. Следуйте инструкциям в Telegram: @poiskkinodev_bot
4. Скопируйте ваш API ключ

## Шаг 2: Запустите синхронизацию

```bash
cd movieswipe-pwa/scripts
node poiskkino-sync.cjs --api-key YOUR_API_KEY
```

Скрипт загрузит до 50,000 фильмов (200 запросов × 250 фильмов).

## Шаг 3: Проверьте результат

```bash
# Статус синхронизации
node poiskkino-sync.cjs --status

# Размер файла
ls -lh ../public/data/movies-poiskkino.json

# Количество фильмов
cat ../public/data/movies-poiskkino.json | grep -o '"id"' | wc -l
```

## Шаг 4: Загрузите в приложение

1. Откройте приложение
2. Перейдите в **Настройки** → **Источники данных**
3. Включите чекбокс **Кинопоиск**
4. Нажмите **Обновить каталог**

Готово! Фильмы загружены в IndexedDB и работают offline.

## Ежедневная синхронизация

Добавьте в crontab для автоматической загрузки:

```bash
# Каждый день в 3:00 утра
0 3 * * * cd /path/to/movieswipe-pwa/scripts && node poiskkino-sync.cjs --api-key YOUR_KEY >> sync.log 2>&1
```

## Что загружается?

- ✅ Рейтинг > 6.0
- ✅ Минимум 1000 голосов (популярные)
- ✅ С постерами
- ✅ Только фильмы (не сериалы)
- ✅ Сортировка по популярности

## Временная стратегия

1. **День 1-2:** Последние 5 лет (2021-2026) → ~50,000 фильмов
2. **День 3-4:** 2016-2026 → еще ~50,000 фильмов
3. **День 5-6:** 2011-2026 → еще ~50,000 фильмов
4. **День 7-10:** До 1900 года → остальные фильмы

## Лимиты

| Параметр | Значение |
|----------|----------|
| Запросов в день | 200 |
| Фильмов за запрос | до 250 |
| Максимум в день | ~50,000 |
| Сброс лимита | 00:00 UTC |

## Полезные команды

```bash
# Загрузить только 10 порций (2500 фильмов)
node poiskkino-sync.cjs --api-key YOUR_KEY --max-requests 10

# Проверить статус
node poiskkino-sync.cjs --status

# Начать заново
node poiskkino-sync.cjs --reset

# Справка
node poiskkino-sync.cjs --help
```

## Troubleshooting

### 403 Forbidden
Превышен дневной лимит. Дождитесь следующего дня.

### 401 Unauthorized
API ключ неверный. Проверьте ключ на https://poiskkino.dev/

### Файл прогресса поврежден
```bash
node poiskkino-sync.cjs --reset
```
