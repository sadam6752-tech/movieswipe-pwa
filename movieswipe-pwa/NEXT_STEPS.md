# Следующие шаги - ПоискКино API

Интеграция с ПоискКино API завершена! Теперь вы можете загружать реальную базу из 1M+ фильмов.

## 🚀 Быстрый старт (5 минут)

### 1. Получите API ключ

```
1. Откройте: https://poiskkino.dev/
2. Нажмите: "Получить API"
3. Telegram: @poiskkinodev_bot
4. Скопируйте ваш API ключ
```

### 2. Запустите синхронизацию

```bash
cd movieswipe-pwa/scripts
node poiskkino-sync.cjs --api-key YOUR_API_KEY
```

**Результат:** До 50,000 фильмов за 2-3 минуты

### 3. Загрузите в приложение

```
1. Откройте приложение
2. Настройки → Источники данных
3. Включите "Кинопоиск"
4. Нажмите "Обновить каталог"
```

**Готово!** Фильмы загружены и работают offline.

## ❓ Troubleshooting

### 403 Forbidden
**Проблема:** Превышен дневной лимит

**Решение:** Дождитесь следующего дня (сброс в 00:00 UTC)

### 401 Unauthorized
**Проблема:** API ключ неверный

**Решение:** Проверьте ключ на https://poiskkino.dev/

### Файл прогресса поврежден
**Проблема:** Ошибка чтения `.sync-progress.json`

**Решение:**
```bash
node scripts/poiskkino-sync.cjs --reset
```

### Нет интернета
**Проблема:** Соединение прервано

**Решение:** Прогресс сохранен, просто запустите снова
