# Система загрузки фильмов по жанрам

## Концепция

Вместо загрузки всех фильмов сразу, приложение использует **ленивую загрузку по жанрам** - загружает только те жанры, которые выбрал пользователь.

## Как это работает

### 1. Первый запуск

При первом запуске приложения:
- Загружается базовый файл `movies.json` (50-100 популярных фильмов)
- Данные сохраняются в IndexedDB
- Пользователь может сразу начать работу

### 2. Выбор жанра

Когда пользователь выбирает жанр в фильтрах:
1. Проверяется, загружен ли этот жанр (`isGenreLoaded()`)
2. Проверяется, есть ли фильмы этого жанра в базе (`hasMoviesOfGenre()`)
3. Если жанр не загружен - автоматически загружается соответствующий JSON
4. Показывается индикатор "Загрузка жанров..."
5. Фильмы добавляются в IndexedDB
6. Жанр помечается как загруженный в настройках
7. Фильтрованный список обновляется

### 3. Повторный выбор

При повторном выборе того же жанра:
- Данные берутся из IndexedDB (мгновенно)
- Дополнительная загрузка не требуется

### 4. Offline режим

После загрузки жанра:
- Все данные доступны offline
- Постеры загружаются по требованию при просмотре
- Service Worker кэширует постеры для offline доступа

## Преимущества

### Производительность
- ✅ Быстрая первая загрузка (~50 KB вместо 5+ MB)
- ✅ Загружается только нужный контент
- ✅ Мгновенный доступ к загруженным жанрам

### Масштабируемость
- ✅ Можно иметь тысячи фильмов без замедления
- ✅ Легко добавлять новые жанры
- ✅ Простое обновление контента

### Пользовательский опыт
- ✅ Приложение готово к работе сразу
- ✅ Прозрачная загрузка в фоне
- ✅ Индикаторы прогресса
- ✅ Работает offline после загрузки

## Технические детали

### syncService

```typescript
// Загрузить один жанр
await syncService.loadGenreFromLocalJSON('comedy');

// Загрузить несколько жанров
await syncService.loadGenresFromLocalJSON(['comedy', 'action', 'drama']);
```

### settingsStore

```typescript
// Проверить, загружен ли жанр
const isLoaded = settingsStore.isGenreLoaded('comedy');

// Добавить загруженный жанр
settingsStore.addLoadedGenre('comedy');

// Добавить несколько жанров
settingsStore.addLoadedGenres(['comedy', 'action']);
```

### useGenreLoader Hook

Автоматически загружает недостающие жанры:

```typescript
const { isLoading, error } = useGenreLoader(selectedGenres);
```

## Генерация файлов

Для генерации файлов по жанрам используйте скрипт `generate-genre-files.js`:

```bash
# Установить зависимости
npm install

# Запустить генерацию
node scripts/generate-genre-files.js
```

Скрипт:
1. Подключается к TMDB API
2. Загружает фильмы по каждому жанру
3. Создает отдельные JSON файлы
4. Сохраняет в `public/data/`

## Размеры файлов

| Жанр | Фильмов | Размер файла | Время загрузки (3G) |
|------|---------|--------------|---------------------|
| Базовый | 50 | ~54 KB | <1 сек |
| Comedy | 500 | ~540 KB | 1-2 сек |
| Action | 500 | ~540 KB | 1-2 сек |
| Drama | 500 | ~540 KB | 1-2 сек |
| **Всего** | **4000+** | **~4.5 MB** | **По требованию** |

## Обновление контента

### Вручную
1. Отредактировать соответствующий JSON файл
2. Пересобрать приложение
3. Задеплоить

### Автоматически
1. Запустить скрипт генерации
2. Скрипт обновит все файлы из TMDB
3. Пересобрать и задеплоить

## Лучшие практики

### Размер файлов
- Держите файлы до 1 MB (до 1000 фильмов)
- Для больших коллекций разделяйте на подкатегории

### Кэширование
- Service Worker автоматически кэширует JSON файлы
- Постеры кэшируются по требованию
- Очистка кэша при обновлении версии

### Обработка ошибок
- Показывайте понятные сообщения об ошибках
- Предлагайте повторную попытку
