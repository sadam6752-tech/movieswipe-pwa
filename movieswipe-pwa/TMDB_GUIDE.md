# Руководство по интеграции с TMDB API

## Что это?

MovieSwipe PWA теперь поддерживает загрузку полной базы данных фильмов из The Movie Database (TMDB) - одной из крупнейших баз данных о фильмах в мире.

## Зачем это нужно?

Вместо 50 предустановленных фильмов вы получите доступ к:
- **До 1000 популярных фильмов** с актуальными данными
- **Автоматическое обновление** информации о фильмах
- **Полные данные**: постеры, описания, актеры, режиссеры, рейтинги
- **Поиск по всей базе** - находите любой фильм из загруженной коллекции

## Как получить API ключ?

### Шаг 1: Регистрация
1. Перейдите на [themoviedb.org](https://www.themoviedb.org/signup)
2. Нажмите "Sign Up" (Регистрация)
3. Заполните форму регистрации
4. Подтвердите email

### Шаг 2: Запрос API ключа
1. Войдите в свой аккаунт
2. Перейдите в [Settings → API](https://www.themoviedb.org/settings/api)
3. Нажмите "Request an API Key"
4. Выберите "Developer" (для личного использования)
5. Примите условия использования
6. Заполните краткую форму о вашем проекте:
   - **Application Name**: MovieSwipe PWA
   - **Application URL**: http://localhost (или ваш домен)
   - **Application Summary**: Personal movie discovery app
7. Получите ваш API ключ

### Шаг 3: Использование в приложении
1. Откройте MovieSwipe PWA
2. Перейдите в **Настройки** (иконка шестеренки)
3. Найдите раздел "Источники данных"
4. Включите чекбокс **IMDB / TMDB**
5. Появится поле для ввода API ключа
6. Вставьте ваш API ключ (формат: `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`)
7. Нажмите **Сохранить**
8. Нажмите **Обновить каталог**

## Процесс загрузки

После нажатия "Обновить каталог":

1. **Прогресс-бар** покажет процесс загрузки
2. Приложение загрузит до 1000 популярных фильмов
3. Для каждого фильма загружаются:
   - Название (русское и оригинальное)
   - Описание на русском языке
   - Год выпуска
   - Жанры
   - Рейтинг
   - Длительность
   - Режиссер
   - Топ-5 актеров
   - URL постера и фонового изображения

4. **Время загрузки**: ~15-20 секунд для 1000 фильмов (всего ~50 API запросов)
5. **Постеры загружаются по требованию** - когда вы просматриваете фильм
6. **Данные сохраняются локально** - работают offline после загрузки
7. **Детали загружаются автоматически** - при открытии страницы фильма (режиссер, актеры, длительность)
8. **Детали кэшируются** - повторная загрузка не требуется

## Что дальше?

После загрузки вы можете:

### 1. Свайп-интерфейс
- Листайте фильмы свайпом
- Свайп вправо = добавить в избранное
- Свайп влево = пропустить

### 2. Поиск по настроению
- Выберите настроение (романтика, адреналин, философия и т.д.)
- Получите случайный фильм, соответствующий вашему настроению
- Жанры автоматически маппятся на настроения

### 3. Каталог
- Просматривайте все загруженные фильмы
- Используйте поиск по названию, актерам, режиссеру
- Фильтруйте по жанрам, рейтингу, длительности

### 4. Избранное
- Все фильмы, добавленные свайпом вправо
- Сортировка по дате добавления, названию, рейтингу
- Просмотр в виде сетки или списка

## Технические детали

### Маппинг жанров
TMDB использует свою систему жанров, которая автоматически конвертируется в жанры приложения:
- Action → action
- Comedy → comedy
- Drama → drama
- Horror → horror
- Romance → romance
- Sci-Fi → sci-fi
- Thriller → thriller
- И т.д.

### Маппинг настроений
Жанры автоматически маппятся на настроения:
- Action → adrenaline, energize
- Comedy → comedy, light-positive
- Drama → emotional, philosophical
- Horror → horror, adrenaline
- Romance → romance, emotional
- Sci-Fi → fantasy, philosophical
- И т.д.

### Ограничения API
- **Rate Limit**: 40 запросов в 10 секунд
- **Начальная загрузка**: ~50 запросов для 1000 фильмов
- **Детали по требованию**: 1 запрос при открытии фильма (кэшируется)
- **Бесплатный план** достаточен для личного использования

### Хранение данных
- **IndexedDB**: метаданные фильмов (названия, описания, актеры)
- **LocalStorage**: настройки и API ключ
- **Постеры**: загружаются по требованию с серверов TMDB
- **Offline-режим**: все данные доступны без интернета после загрузки

## Безопасность

- API ключ хранится локально в вашем браузере
- Ключ не передается на сторонние серверы
- Используйте ключ только в личных целях
- Не публикуйте ключ в открытом доступе

## Обновление базы

Вы можете обновить базу фильмов в любое время:
1. Перейдите в Настройки
2. Нажмите "Обновить каталог"
3. Старые данные будут заменены новыми

## Проблемы и решения

### "Неверный API ключ"
- Проверьте, что скопировали ключ полностью
- Убедитесь, что используете API Key (v3 auth), а не Read Access Token
- Проверьте, что ключ активирован в настройках TMDB

### "Ошибка загрузки"
- Проверьте интернет-соединение
- Подождите несколько минут и попробуйте снова
- Возможно, превышен rate limit - подождите 10 секунд

### "Загрузка слишком долгая"
- Загрузка 1000 фильмов занимает ~15-20 секунд
- Прогресс-бар показывает текущее состояние
- Не закрывайте приложение во время загрузки

### "Нет постеров"
- Постеры загружаются по требованию при просмотре
- Проверьте интернет-соединение
- Некоторые старые фильмы могут не иметь постеров в TMDB
