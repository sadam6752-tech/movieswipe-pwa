  - Интеграция с общей системой фильтров
- **Пагинация в каталоге** - показываем по 100 фильмов с кнопкой "Показать ещё"
  - Быстрая загрузка страницы (200-300ms вместо 2-3 секунд)

  - Плавная прокрутка без лагов

  - Счетчик оставшихся фильмов
- **Интеграция с ПоискКино API** - инкрементальная синхронизация с базой 1,085,549 фильмов
  - Скрипт `scripts/poiskkino-sync.cjs` для загрузки фильмов из ПоискКино API
  - Умные фильтры (рейтинг > 6.0, популярные, с постерами, только фильмы)
  - Автоматическое отслеживание лимита (200 запросов/день)
  - Прогресс сохраняется в `.sync-progress.json` (можно остановить и продолжить)
  - Временная стратегия (последние 5 лет → расширение на 5 лет назад)

  - Маппинг жанров → настроения (автоматически)
  - Сортировка по популярности (количество голосов)
  - Курсорная пагинация (v1.5 API)

### 🎯 Технические детали
- **API:** https://api.kinopoisk.dev/v1.5/movie
- **Аутентификация:** X-API-KEY header
- **Лимит:** 200 запросов/день (бесплатно)
- **Фильмов за запрос:** до 250
- **Максимум в день:** ~50,000 фильмов
- **База данных:** 1,085,549 фильмов (Кинопоиск + TMDb + IMDb)
- **Язык:** Русский (названия, описания, актеры)
- **Рейтинги:** КП, IMDB, TMDB
- **Временная стратегия:** 2021-2026 → 2016-2026 → 2011-2026 → 1900-2026
- **Прогресс:** Сохраняется локально, можно продолжить в любой момент
- **Автоматизация:** Поддержка cron и GitHub Actions

## [1.1.0] - 2026-03-07

### ✨ Добавлено
- **Интеграция с TMDB API** - загрузка до 1000 популярных фильмов
- **Гибридный подход к загрузке** - базовая информация сразу, детали по требованию
- **Ленивая загрузка деталей** - режиссеры, актеры и длительность загружаются при открытии фильма
- **Кэширование деталей** - детали сохраняются в IndexedDB для offline доступа
- **API ключ в настройках** - безопасное хранение ключа TMDB
- **Прогресс-бар загрузки** - визуальная индикация процесса загрузки
- **Индикатор загрузки деталей** - бейдж "Загрузка деталей..." на странице фильма
- **Автоматический маппинг жанров** - конвертация жанров TMDB в систему приложения
- **Автоматический маппинг настроений** - жанры автоматически связываются с настроениями
- **Ленивая загрузка постеров** - изображения загружаются по требованию
- **Быстрая загрузка** - 1000 фильмов за ~15-20 секунд

### 🎯 Технические детали

- Rate limiting: 250ms между запросами к TMDB API
- Retry механизм с exponential backoff
- Прогресс-колбэки для UI обновлений
- Маппинг 18 жанров TMDB на 17 жанров приложения
